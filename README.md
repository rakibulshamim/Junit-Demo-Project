# Junit-Demo-Project

## Technology:
- Tool: selenium
- IDE: IntelIJ
- Build tool: Gradle
- Language: Java
- Framework: JUnit


## Prerequisite:
1. Need to install jdk 1.8 or above
2. Need to setup JAVA_HOME and GRADLE_HOME


## How to run this project:
1. Clone the project
2. Open terminal inside the project folder
3. Give following command:
```
gradle clean test
```
